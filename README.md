# Restaurant Antopolis Task Client

This is the frontend application for the Restaurant Antopolis Task.It is fully built with Next.js 16, TypeScript, TailwindCSS, and Framer Motion, providing a modern UI with smooth animations.
The app includes features like dish management, category creation, home UI, members, feedback, and contact section.
